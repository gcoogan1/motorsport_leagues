# motor_sport_leagues
**MotorSport Leagues** is a web platform for Gran Turismo 7 and iRacing drivers to create, manage, and follow racing leagues. The goal is to make league discovery, participation, and organization easier across multiple sim racing platforms.

## Teck Stack

### Frontend
- **React**
- **TypeScript**
- **styled-components**

### Backend
- **Supabase**
  - Authentication
  - PostgreSQL
  - Row Level Security
